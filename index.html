Approved Mods
These are the only mods allowed on the server. Some mods unlock over time as the server progresses. Players are responsible for following the unlock schedule.

How to Install Mods

Download the mod files and place them in your Factorio mods folder:

  • Windows: %appdata%\Factorio\mods\
  • Download the .zip files from the links below
  • Copy the .zip files directly to the mods folder (don't extract them)
  • Restart Factorio if it's running

Approved Blueprints
Only blueprints from this list are allowed. We want everyone to experience building and problem-solving together, so we've limited it to blueprints that help with specific technical challenges rather than complete base designs.

How to Import Blueprints

  • Click "Copy Blueprint String" button below
  • In Factorio, open your inventory (E key)
  • Click the Blueprint Library button (book icon)
  • Click "Import string" button
  • Paste the blueprint string (Ctrl+V) and confirm
